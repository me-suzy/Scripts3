phpSiteCheck v.1.0


1/ How to install this tool to your server?

Just perform the following easy steps:
 1. Unpack phpSiteCheck.zip to your website folder
 2. Configure settings at the bottom of sitecheck.inc.php if necessary
 3. Add the following PHP code into your PHP page code:
    include("./sitecheck.inc.php");
			 
Also you can check our example.php script.
