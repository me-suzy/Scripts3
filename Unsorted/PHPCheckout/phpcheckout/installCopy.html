Install Helper

Copy files from the Distribution Package

The Install Helper needs you to copy files from the distribution package onto your web server. Below are the steps to take:

  1. if you are reading this then you have already unzipped the distribution file and you already have a phpcheckoutTM folder and file structure
  2. IF you have unzipped to a location which is not your web server, then Install Helper needs you to move or copy your unzipped phpcheckout folder and file structure to your web server
  3. you will now have a copy on your web server
  4. you need only the copy on your web server
  5. browse phpcheckout/installCopy.html - with your browser from the web server only.

To continue, please click on the "Next" button from the web server only.
