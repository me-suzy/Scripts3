Glossary

This has definitions of variables, CONSTANTS, labels or features used in phpCheckoutTM.

Glossary of Product Features

Resource Type
the type of digital product
Availability
shows if the product is free, retail or other.
License
tells the general restrictions on use of the product.
Platform
the operating system the product runs on.
Language
tells the name of the script language used to create the product
Category
the general slot or type of product
Newest Release Date
the date the product was released or last revised
File Size
The size of the product. Actual standard measurement unit (example kilobytes) is determined for each product by the administrator inserting or updating the product
Version
A unique control unit associated with the product instance
Product #
An internal number which uniquely identifies a specific product record
