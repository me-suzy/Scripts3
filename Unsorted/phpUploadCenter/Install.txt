English
~~~~~~~

1. Create folder for php script on the server.
2. Customize, values of variables in file include/conf.php
   See comments in file.
3. Customize files nn.php (where nn - is language code).
4. Write in your text in all html-files.
5. Copy all files and subfolders to server.
6. Register your own administrator account.
7. Activate your new account (needed, if request activation is "on")
8. Log in as user "Admin", password "test".
9. Go to the administrator console and set to the new account an administrator rights.
10. Log in using new account, go to the administrator console and delete "Admin" account.
    DO NOT FORGET DO THIS!!!  If forget - this is serious security hole!
11. Be sure, that user accounts is not visible, when you typing in browser the address www.myserver.com/upload/users/username.
   (where www.myserver.com - is the server name where you installed your script, username - the name of any registered user)
   If visible, you have to configure server!
