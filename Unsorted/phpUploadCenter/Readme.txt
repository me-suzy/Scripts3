English
~~~~~~~

This is PHP script of the the File Upload Center.
It has been wrote for my own site and
works on Windows/Linux + Apache + PHP 4.0.3.
Main features:
1. Multilangual (English, Russian, Polish, German).
2. Time zone offset (depends on selected language).
3. User can give description to file.
4. Download counter.
5. User can sort files by name, upload date, rating.
6. Administrator can show own messages.
7. Filename length limit.
8. Autodeleting files older than NN days.
9. Appearance tuning (colors, font)
10. Is is possible to define header and footer.
11. Administrator account to delete files
12. Possibility to view contents of ZIP archives without downloading.
Appeared in 2.x versions:
13. Now it is possible view files for certain date.
14. User acccounts support.
15. IP Logging.
16. Account management system.
17. In order to check user e-mail addresses you can enable user activation system.
18. User can subscrbe on everyday digest of uploaded files list via e-mail (optional).
19. Users can get small files by e-mail (optional).
20. Skins support.
21. User statistics (uploaded/downloaded/e-mailed files, last access).
22. Administrator can show info message over the file list.
