If you ship your products via UPS, then run this file on your server. It is a tiny little object that enables your server to communicate in real-time with the UPS.com servers. The UPS.com servers will then be able to send rate information to your checkout page.

Step 1: Double-click on the UPSCalcPro.exe file.

Step 2: You're done! That's all you have to do to setup UPS on your server.
