In the configuration form:

  • Enter the symbol used for your currency. This is the actual symbol not the HTML code.
    Examples:  £    $    €    Lire    ¥    DM
  • Select the location for the currency symbol, either before or after the number.
  • Select the format for your currency display. (See the list below).
