Settings
  • My Details

    my url - url of your site.
    Should specify index.php file (for example http://mysite.com/index.php). In case if index.php is installed as index of directory (for example such line DirectoryIndex index.php in .htaccess file) it is possible to specify directories url (for example http://mysite.com/).
    my title - the name of site.
    my email, my icq - the contact data is displayed in webmaster's interface.
    extra url - url where exits of overflow should be sent.

  • Script Settings

    authentification - recognition of belonging to trade
    The script can work in 2 modes:

    - enable ID authentification - recognition of belonging to trade by value from transmitted in url at reference to a site and by referer value transmitted by browser.
    - referer authentification only - recognition of belonging to trade only by referer value transmitted by browser, from value transmitted in url at reference to a site is ignored.

    recalculate time (seconds) - time interval between recalculations.

    unique time (IP-log time) (seconds) - storage time of IP in IP-log.

    refering URLs - to perform or not to perform refering URLs log

  • In Pages

    file extension - extension of entrance pages (without a dot)
    method - method of entrance pages

  • Default Group

    - trade group by default. It is used in other adjustments.

  • TGP & Real Content Features

    default out to url(hurl) percent - percent of exits to real content by default.

    default first time click - adjustment of the "first click":
    out to real content - "first click" always goes to content
    use percent - not to process "first click" in special way

    default tracking of first time click to url(hurl):
    track - consider any first exit to content as the first one - if option of exit gtr=0 is not set
    don't track - not to consider any first exit to content as the first one - if option of exit gtr=1 is not set

  • Default Out - default values, when there is no options of exit

    group
    default group - exit by default only to sites with number of group by default
    custom groups - exit by default only to sites of the specified groups (groups are divided - symbol)

    target
    any site - exit by default to sites of any type
    pictures site - exit by default only to sites such as "only pictures" or "mixed"
    movies site - exit by default only to sites such as "only movies" or "mixed"

  • New Trades Defaults - adjustments of new trades, registered through webmaster's interface.

    group
    default group - new trade will be appropriated to default group, hour schedule of groups of new trade is filled by number of group by default
    webmaster should choose the group (non-blank fields) - webmaster should choose the group manually. Here it is necessary to define, which groups will be accessible to the webmaster for choice and specify names of these groups. Names of groups inaccessible for webmaster to choose are necessary to leave clear.

    in pages prefix - prefix of entrance page compared to the new trade depending on its group.

    site type
    webmaster can choose - webmaster chooses the type of his site
    set to - rigid comparison of type of site to the new trade

    default trade type - type of trade established for new trade

    default ratio - ratio, established for new trade

    default hourly force - this value fills the hourly schedule of forced traffic of new trade

    default daily in trigger - established for new trade

    default daily out trigger - established for new trade

    default previous hour in trigger - established for new trade

    default previous hour out trigger - established for new trade

    include new trade in top list - whether to include new trade to toplist

  • Top List - toplist settings

    members maximum - quantity of toplist members
    If to set quantity of toplist members to 0 - toplist is not formed
    order - the order of sorting sites in toplist

  • Webmaster's Interface Control

    signup form - permission or prohibition of establishment of new trades through the webmaster's interface
    webmaster's stats - permission or prohibition for the webmaster to view status of trade on your site

    In a case if the webmaster can view status of trade on your site:
    webmaster's stats raw in - whether to show the raw ins to the webmaster
    webmaster's stats unique in - whether to show the unique ins to the webmaster
    webmaster's stats clicks - whether to show the quantity of clicks, which exits have gone to trade
    webmaster's stats gallery clicks - whether to show the quantity of clicks, which exits have gone to real content
    webmaster's stats productivity - whether to show the productivity of trade

  • Admin Interface Control

    use special group colors - here it is possible to set color of display of trade in main table according to its group. Color is specified in #ffffff format.
