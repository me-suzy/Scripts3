upgrade from phpMyTGP ver.0.3b to ver.0.4b

Replace files from archive with update.
Run update_to_v4.php from phpmytgp/install directory
or alter table mytgp_set from your MySQL base manually by update_to_v.0.4.sql

WHATS NEW v.0.4

  • Fixed some old bugs
  • Added some new bugs
  • Possibility to limit daily submissions number from one webmaster per category, per domain, per email and total separately for trusted and regular webmasters
  • Now your can work with VIP galleries submited from trusted webmasters:
       - Add new galleries from addvip.php file
       - New Menu added VIP Setup Section to admin script
       - Don't forget check USE VIP POSTS? in Settings for working with VIP galleries
       - You can check VIP galleries anyway from template file vip.tpl
       - You can place VIP galleries first in list by checking Stay VIP First? in Settings Section of Admin Menu
       - Separatly setup rotate out days for VIP galleries and other posts
       - New template file vip.tpl describe how to mark VIP galleries and new template field for link.tpl <!--vip--> show this mark in the list of all galleries
  • When you add new rule to blacklist program can delete all existing posts with this rule from base
  • Settings to forbid new window openning, status line change, onClick, onMouseover scripts
