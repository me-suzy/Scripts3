phpMyTGP Manual.

About some Settings fields

  • Admin password - Change and Remember This Field!
  • Admin Mail - Your email address
  • TGP Name - This is what your TGP site will be called
  • Language file - show content of directory \phpMyTGP\language\ with language templates.
  • PhpMyTGP Setup Dir - The server path to your phpMyTGP files ( example : /home/you/public_html/phpMyTGP/ ) WITH TRAILING SLASH
  • TGP Main FileName - This is the name of the main file that will be created as your main tgp page. The extention of this file will be used for all sub pages created from templates as well
  • TGP Main Dir - The server path to your TGP html directory ( example : /home/you/public_html/tgp/ ) WITH TRAILING SLASH
  • TGP Main URL - The URL to your phpMyTGP html directory. ( example : http://www.yoursite.com/tgp/ ) WITH TRAILING SLASH
  • Path to SENDMAIL - The server path to your mail program. ( example : /usr/sbin/sendmail ) NO TRAILING SLASH
  • Length of LIST template - amount of lines which will be displayed in html files if the command <! -- list --> is applied in templates
  • Auto validate post - Set program to auto validate all posts
  • Send E-Mail - Send user an email after he signs up with content of template mailwelcome.tpl and when user post rotated out with content of template mailout.tpl
  • Precheck postings - Set program to try check postings (linkback and server response) before its applied.
  • Max Post per Domain - Maximum posting allow per domain name
  • Max Post per E-Mail - Maximum posting allow per E-Mail
  • Rebuild Time - amount of time in minutes after which the program will update contents of all pages
  • Rotate Days - amount of time in days after which the program will delete old galleries from your base
  • Check Phrase - Phrase (usually from linkback to your tgp) on which presence will be checked all galleries ( example : yoursite.com/tgp/ )

Templates
phpMyTGP creates all pages from a series of template files from /phpMyTGP/templates/ directory. This enables the webmaster to have complete control of the entire site layout and design. Each template has a series of tags that can be used to place specific script generated content. If this seems confusing just look at included template files. It is really VERY simple and flexible.

  • VIP Template
     Template file - vip.tpl describe how mark VIP galleries in the list.
  • Link Template
    There is one template file - link.tpl for formatting the link on posted gallery.

    You can use these Template Tags :
    <!--date-->      : This prints the date of posting of gallery.
    <!--catname-->   : This prints the category of gallery.
    <!--desc-->      : This prints the description of gallery.
    <!--images-->    : This prints the number of images in gallery.
    <!--vip-->       : This prints VIP mark how described in vip.tpl
    <!--url-->       : This prints the url of gallery.
    for example this template:
    <!--date--> - <!--catname--> - <!--images--> thumbs of<a href= "<!--url-->"<!--desc--></a><BR>
    will show like this:
    12 Sep 2002 - MENS - 15 thumbs of cool mens

  • Category Templates
    There are two default files: default.tpl and categories.tpl
    - default.tpl -     This file is used for the main "overall" category this will always be used for the main index page
    - categories.tpl - This file is the default template for all categories

    - Separate Category Files - You can make a separate template file for each category. The template files will be named according to the category names they must be in all lower and can only contain letters & numbers. So if you had a category called Mens the template file you would create would be called mens.tpl.

    In both of the above cases you have the ability to use the following tags in the templates. These tags will be replaced with the appropriate content when the html is generated. Remember when placing these tags that they are all lower case.

    You can use these Template Tags :
    <!--join-->    : This place a link directly to the submission form.
    <!--terms-->   : This place a link to the terms and condition form.
    <!--navbox-->  : This generates a drop down nav box to every category.
    <!--link-->    : This prints the one link to submited gallery.
    <!--list-->    : This outputs the list of N galleries, where N setup in settings.

    <!--list|CATNAME|NUMPOST|--> : ONLY FOR default.tpl !!! Show NUMPOST postings from category CATNAME on main page of your TGP.
    <!--catname--> : ONLY FOR categories.tpl or separate category files!!! This prints the category of curent TGP page.

  • Terms Template
    This template is very simple, it is a standard HTML file called terms.tpl with terms and condition of your TGP
    You can use these Template Tags :
     <!--join-->   : This place a link to the submission form.

  • New Gallery Submission Template
    The new.tpl file controls the layout of the gallery submission form.
    You can use these Template Tags :
    <!--start--> : This tag is required and starts the submission form.
    <!--email-->  : This tag places a box for the uses to input his email.
    <!--cats-->   : This tag places a drop down box of available categories.
    <!--pics-->   : This allows the user to select the number of pics in his gallery.
    <!--url-->    : This tag places a box for the users gallery URL.
    <!--desc-->   : This tag places a box for the users gallery description.
    <!--mlist-->  : This tag places a check box that allows the user to be added to your mailing list.
    <!--finish--> : This is required & finishes the submission html form

  • Post template
    The post.tpl file controls the page that is shown to the webmaster after completing the sign up process.
    You can use these Template Tags :
    <!--email-->  : This tag show user submited email.
    <!--cats-->   : This tag show user submited category.
    <!--pics-->   : This show the number of pics in his gallery.
    <!--url-->    : This tag show the submited gallery URL.
    <!--desc-->   : This tag show the submited gallery description.
    <!--link-->   : Curently not in use

  • Mail templates
    The mailwelcome.tpl , mailout.tpl and maildelete.tpl files is the email messages that are sent to the user after they submit there gallery, after there gallery was rotated out and after there gallery was not approved.
    You can use these Template Tags :
    <!--email-->  : This tag show user submited email.
    <!--catname-->: This tag show user submited category.
    <!--pics-->   : This show the number of pics in his gallery.
    <!--url-->    : This tag show the submited gallery URL.
    <!--desc-->   : This tag show the submited gallery description.
    <!--tgpname-->   : This tag show the name of your TGP.
    <!--link-->   : Curently not in use
