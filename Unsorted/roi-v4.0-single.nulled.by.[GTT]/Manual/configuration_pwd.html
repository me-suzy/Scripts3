  Password configuration  
The script is protected with up to 20 symbols password.
To change the password, please fill the following fields in password configuration section of configuration form:
Old password – current script admin area access password. If you are setting the password for the first time leave this field blank.
New password – new script admin area access password
Repeat new password – to avoid password misspelling please repeat the new script admin area access password here.
After filling in these fields click the Change button to accept password change.
