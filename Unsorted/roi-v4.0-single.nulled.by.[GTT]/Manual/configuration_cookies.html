  Cookies configuration  
When you create a new action for your campaign the script generates the code fragment that should be included in html code of your web page, to track visitors by using cookies. In this section you could set up the expiration time for these cookies by filling in the following field
Cookie expiration time – when the time expires, the cookie will be rewritten by new data or simply deleted from your visitor’s hard disk.
