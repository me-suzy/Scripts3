About Form Mail: eMail Form Processor Pro

Form Mail: eMail Form Processor Pro is the script that can be used to process any forms on your web site. The general use of this script is feedback
forms but it can be used in a huge variety of other ways. Here are just a few features of this great product:

It’s easy to install and customize
Works with any form
You have full layout and design control
It can process Multiple Pages forms
Works with email and user attachments
One script can process any amount of forms
