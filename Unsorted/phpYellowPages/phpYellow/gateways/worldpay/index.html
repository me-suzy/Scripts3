How to Make Worldpay Work
with phpYellow Pages

Pretest

Before changing anything make sure that your phpYellow Pages is processing paid listings. This is how:

  1. open util.php
  2. set the CONSTANT value for BYPASSCARDPROCESSOR from "no" to "yes"
  3. set the CONSTANT value for PAIDLISTINGS from "" to "yes"
  4. save util.php to your web server
  5. go to phpYellow Pages and add a test listing
  6. find any test listing
  7. click on the "Go Premium" button
  8. complete the checkout page details and click on "Next Step"
  9. on the Proceed to Purchase? page click on "Next Step"
  10. the result is an upgraded Premium Listing
DO NOT PROCEED UNTIL THIS PRETEST IS WORKING TO YOUR SATISFACTION.

phpYellow Pages can use Worldpay to process Premium Listing payments. A small amount of configuration is needed on the Worldpay Administration System (WAS) to make basic settings. Also, phpYellow Pages must be configured as below:

Makes these changes on Worldpay Administration System (WAS)

https://select.worldpay.com/wcc/admin

Make these changes to WAS - Installations - Configuration options :

  1. Description : phpYellow Pages
  2. Customer description : Service - Premium Listing
  3. Merchant's shop URL : http://www.yourdomain.com/phpYellow/yellowresult.php
  4. Callback URL : http://www.yourdomain.com/phpYellow/premiumProcessPayment.php
  5. Callback enabled? Yes, check this checkbox
  6. Leave all other configuration options on this page unchanged

Make these changes to /phpYellow/gateways/worldpay/resultY.html

  1. open the file, find the form action
  2. change the form action to be action="http://www.yourdomain.com/phpYellow/new.php"
    Example: http://www.dreamriver.com/phpYellow/new.php

Make these changes to WAS - Installations - File Management :

  1. upload resultY.html

Makes these changes to phpYellow Pages

In the file called util.php make these changes:
  1. define("PAIDLISTINGS", "yes");
  2. define("SENDRECEIPT", "yes"); /* [optional - WAS can be configured to send an email receipt] */
  3. define("MERCHANTACCOUNT", "Your Worldpay Installation ID");
  4. define("BYPASSCARDPROCESSOR", "no");
  5. define("GATEWAYPROCESSOR", "worldpay");
  6. define("POSTTRANSACTIONSURL", "https://select.worldpay.com/wcc/purchase");
  7. save and upload to your webserver

Testing

Make a few test listings to make sure everything works the way you want it to.

Done!

This completes the steps to integrate your payment processing with phpYellow Pages. Congratulations, you can start earning revenue using online payment and automatically paid listings.
