with phpYellow Pages

Pretest

Before changing anything make sure that your phpYellow Pages is processing paid listings. This is how:

  1. open util.php
  2. set the CONSTANT value for BYPASSCARDPROCESSOR from "no" to "yes"
  3. set the CONSTANT value for PAIDLISTINGS from "" to "yes"
  4. save util.php to your web server
  5. go to phpYellow Pages and add a test listing
  6. find any test listing
  7. click on the "Go Premium" button
  8. complete the checkout page details and click on "Next Step"
  9. on the Proceed to Purchase? page click on "Next Step"
  10. the result is an upgraded Premium Listing
DO NOT PROCEED UNTIL THIS PRETEST IS WORKING TO YOUR SATISFACTION.

OVERVIEW

To integrate a ClickBank account with phpYellow Pages Premium Edition (for realtime payments):

a) the phpYellow "buy" page "premiumBuyForm.php" needs to be custom tailored. A custom tailored ClickBank buy page is available in the /clickbank folder. Copy it into the /phpYellow folder, overwriting the file that is there.

b) the receiving url "premiumProcessPayment.php" must be identified as the thank you page in your ClickBank administration, like this:

http://www.yourdomain.com/phpYellow/premiumProcessPayment.php

Note: the Thankyou page link in your clickbank administration will NOT work. The phpYellow thank you page is dynamic. It needs to receive certain variables that are dynamically generated - like those made when purchasing a Premium listing. Examples of these variables include ckey, item, duration... It is because these dynamic variables are not present by simply clicking the link to the thank you page that they do NOT work on your ClickBank administration page.

c) you must enter your clickbank nickname next to CLICKBANK_NICKNAME in util.php. Save and upload to your web server.
