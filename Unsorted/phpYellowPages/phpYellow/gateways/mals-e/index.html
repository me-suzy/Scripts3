How to Make mals-e.com Work
with phpYellow Pages

Pretest

Before changing anything make sure that your phpYellow Pages is processing paid listings. This is how:

  1. open util.php
  2. set the CONSTANT value for BYPASSCARDPROCESSOR from "no" to "yes"
  3. set the CONSTANT value for PAIDLISTINGS from "" to "yes"
  4. save util.php to your web server
  5. go to phpYellow Pages and add a test listing
  6. find any test listing
  7. click on the "Go Premium" button
  8. complete the checkout page details and click on "Next Step"
  9. on the Proceed to Purchase? page click on "Next Step"
  10. the result is an upgraded Premium Listing
DO NOT PROCEED UNTIL THIS PRETEST IS WORKING TO YOUR SATISFACTION.

phpYellow Pages can use mals-e.com to process Premium Listing payments. Configuration is needed in one (1) place:

  1. configure phpYellow Pages

1. configure phpYellow Pages Premium Edition

In the file called util.php make these changes:

  • define("PAIDLISTINGS", "yes");
  • define("SENDRECEIPT", "yes"); /* [optional] */
  • define("BYPASSCARDPROCESSOR", "no");
  • define("GATEWAYPROCESSOR", "mals-e");
  • define("POSTTRANSACTIONSURL", "http://www.aitsafe.com/cf/pay.cfm");
  • save and upload to your web server

In the file called /phpYellow/gateways/mals-e/premiumBuyForm.php make these changes:

  • On about line 64 enter the value for your userid:

    <INPUT TYPE="HIDDEN" NAME="userid" VALUE="1234567">
  • save and upload to your web server

Test

Do a live credit card payment transaction from any phpYellow listing (similar to the Pretest above)

Note

Completing this process also requires configuring your mals-e.com administration area. You will have to do this on a one by one basis - there are a number of variable options for payment processing. There is no one set way to bring in your processor. Also, this method leaves the updating to a manual process. Although the customer's payment can be accepted, their listing still needs to be manually updated using the point and click 'Set CKEY Parameters' tool found in phpYellow Pages Premium Edition Admin Center.

This completes the steps to integrate your payment processing with phpYellow Pages. Congratulations, you can start earning revenue using online payment and automatically paid listings.
