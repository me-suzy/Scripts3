How does phpYellow Pages work?
phpYellow Pages displays yellow pages to visitors. The listings are added by visitors to YOUR website. The listings are unique for each website. The overall purpose of the yellow page directory may be customized by you. You may have just geographic data, or just industry specific listings, or a list of associations ... the variations are endless.

What does phpYellow Pages need to run?
phpYellow Pages requires any web server running php4 with a mysql database. Both php4 and mysql are free to download. The Standard version of phpYellow Pages is also free. Basically you can have an online customized database directory for free, complete with administration scripts.

Ask your internet service (ISP) provider for a mySQL database to be setup for you. The price is small - about $25 one time labor setup fee. You also need to have php4.

What is the maximum number of listings?
You may have an unlimited number of listings. The largest number of records observed by the author was over 25,000 listings on a UK site. phpYellow Pages in theory should scale well with mysql to several hundred thousand ( 300,000+ ) records. Some developers run in excess of a million rows on mysql. This seems to require hardware and software optimization. It is possible, and has been done.

Can an advertiser list in multiple categories?
Yes. The free version does not support multiple categories. The Premium paid version easily accepts many category listings from one visitor.

Is phpYellow Pages available in a language other than English?
Yes. UNSUPPORTED free versions are downloadable for Italian, Russian, Spanish, German or French. These versions are NONSTANDARD and DO NOT INTEGRATE with the English Standard or English Premium versions. Nevertheless, these other language versions are contributed for the benefit of the internet community. Click here for more info.

Can phpYellow Pages run on a local machine?
Yes. The author runs phpYellow development locally. Set the INSTALLPATH in util.php. Localhost is the default.

Do you have documentation for installation and how to install?
Yes. Documentation for installation is found in two places: the installation script called "install.html" and in the phpYellow Pages Developer's Guide. The Guide is included in the free and paid distribution files and contains the most information about this topic.

How long does it take to install?
An "average" php programmer can install phpYellow Pages in less than 20 minutes. An amateur should be able to install in about 40 minutes total. An installation file makes this quite easy. The installation file checks for php, checks for a mysql database and will automatically create new database tables and insert test records for you. Essentially all you need to do is to configure your util.php file, upload all files to your webserver and run the install program from your web server.

Can I use my own categories?
Yes! phpYellow Pages is designed to be customized - especially the categories. Refer to the phpYellow Pages Developer's Guide customization section for more info.

Will phpYellow Pages show subcategories?
No. Subcategories are not supported. Instead, use the description field to precisely identify the listing. This field is indexed and is easily searched and listings are easily found with the included search tools.

Can phpYellow Pages be used commercially?
Yes. The free version and the Premium version may be used commercially. See the license.
