Install - Configure

Configure your database connection information.

  1. open util-dist.php (or util-dist-Premium.php) into an html or plain text editor
  2. save as util.php
  3. find the section called " /* Database and connectivity */ "
  4. enter your specific values for these orange colored constant values only:
    • define("DBNAME", "yourdatabasename" ); /* replace yourdatabasename with the real name */
    • define("DBUSERNAME", "root" ); /* replace root with your mySQL username */
    • define("DBPASSWORD", "yourmysqlpassword" ); /* enter yourmysqlpassword only if required */
    • define("DBSERVERHOST", "localhost" ); /* replace localhost with the name of your database host server */
    • define("DBTABLE", "contact" ); /* DO NOT CHANGE */
    • define("DBTABLE2", "category" ); /* DO NOT CHANGE */
    • define("INSTALLPATH", "http://www.yourDomain.com/phpYellow/"); /* Your phpYellow URL */
  5. save as util.php
  6. upload the util.php file to your web server into the /phpYellow folder.
  7. browse this installConfigure.html file on your web server /phpYellow folder - not on your local machine.
Hint: If you are having trouble connecting to your database then copy the data above and email it to your internet service provider. Ask them what the correct values should be for DBNAME, DBPASSWORD, DBUSERNAME and DBSERVERHOST. You may also send them this URL: http://www.dreamriver.com/phpYellow/installConfigure.html
