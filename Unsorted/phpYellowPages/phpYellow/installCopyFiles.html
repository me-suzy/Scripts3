Install - Make Folders and Copy Files

Copy phpYellow Pages files from the distribution package to your web server.

  1. uncompress the phpYellow Pages distribution package
  2. copy all files to your web server
Hint: Use a standalone FTP program to drag and drop the entire extracted package to the destination. Superior FTP programs will create the folders and copy the files to the target with one drag operation. You can download a free Windows based FTP program if you need one.
