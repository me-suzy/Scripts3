Installation Instructions

  1. Unzip all files locally into a new folder.
  2. Connect to you web server using and FTP program such as CuteFTP or SmartFTP and upload these files to your public HTML directory. Please maintain the same folder structure as provided in the zip file and overwrite all existing files on your server.
  3. Open conf.xml and add the following line:
    <field type="checkbox" name="free_im" text="Allow nonpaying members to IM?">0</field>
    below:
    <field type="checkbox" name="no_verify_email" text="Skip email verification?">0</field>
  4. Make sure your database settings are correct and run upgrade.php to update the MySQL tables. Delete this file once this step is complete.
  5. You have successfully upgraded webDate to version 1.3.1!
