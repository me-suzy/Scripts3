Installation Instructions

  1. Unzip all files locally in a new folder.
  2. Connect to you web server and upload these files to your public HTML directory. Please maintain the same folder structure as provided in the zip file. Allow your FTP client to overwrite any existing files you have.
  3. Set the file permissions on all of the files and folders you just uploaded to 777 or 755. A common mistake made when installing this script is that not every file is set to 755 or 777.
  4. Using phpMyAdmin, or similar MySQL administration software, execute the contents of dt_zips.sql to create the database tables used by the plugin.
  5. You have successfully installed the zip code plugin!
