Installation Instructions

  1. Please replace billing.pml with the existing file in engine/pages and the other two files with the existing files in the templates folder. Then, open smartway/conf/conf.xml and add the follow lines of code below the 2checkout information so it looks like this:

    <section name="2CHECKOUT.COM CONFIGURATION">
    <field required="1" type="edit" name="checkout_url" text="URL">https://www.2checkout.com/cgi-bin/Abuyers/purchase.2c</field>
    <field required="1" type="edit" name="checkout_id" text="Your ID">16984</field>
    </section>
    <section name="PAYPAL CONFIGURATION">
    <field required="1" type="edit" name="paypalemail" text="PayPal Email">sales@web-scribble.com</field>
    </section>

    In the admin area, you will now be able to configure the PayPal information.
