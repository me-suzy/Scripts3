Installation Instructions

  1. Unzip all files locally in a new folder.
  2. Connect to you web server and upload these files to your public HTML directory. Please maintain the same folder structure as provided in the zip file.
  3. Set the file permissions on all PHP files and conf.xml (in smartway/conf/conf.xml) you just uploaded to 777.
  4. Go to http://yourserver.com/path_to_files/smartway/index.php. Log in with the username

    warezscript

     and the password

    warezscript

    . On the left menu under Plug-ins click System Configurator, here you can set various settings including database settings. If you see MySQL errors at the top of the pages, it is normal for some servers. We will be configuring MySQL shortly.
  5. Under Database Configuration, enter your database address (usually localhost), database name, username, and password. Also, please enter the serial number that was supplied to you with your order. Click Save Configuration once you are finished.
  6. After you are certain that you database login information is correct, go to http://yourserver.com/path_to_files/install.php. This will create your database tables for you. Once this is complete, delete install.php and tables.sql from your server.
  7. If you would like to customize your site's layout, please modify the files in the templates folder. Each file is named according to it's function. For example, billing_history.ihtml is the template for billing history. Edit page_top.ihtml and page_bottom.ihtml to set your site's headers and footers. If you are unclear about any of this, please feel free to contact us.
  8. If you would like to edit any of the email messages, please go to the templates/mail directory. Each file is named according to it's function. The first line of each file is the subject of the message and can be edited to fit your needs.
  9. You have successfully installed webDate! Please read the usage instruction for information on how to insert ads and use the admin area.

Usage Instructions

  1. All new users can access the client area at http://yourserver.com/path_to_files/index.php and can register here.
  2. In the admin area, you can edit the elements of each category. For example, if you would like to add a new body type, click on Body Types and add a new type which will automatically appear on the profile creation page after it is added.
  3. A more detailed documentation can be found at: http://www.webscribble.com/products/webdate/doc.shtml
