Standard Theme

Open the "style.css" file under the \themes\standard directory (or click here). Any text editor will handle this nicely.

The page is marked out by section with */----------Section----------*/ tags. Each section allows for customizing of various parts, however the most important are marked out to the right of the setting with instructions.

Below is a brief overview of the variable you should set:

/*-----------Page Variables-----------*/

background-color: #FFFFFF; /* This will set the color of your page */

width: 100%; /* This will set the overall width of your layout */

background-color: #FFFFFF; /* Color of the box's at the top and bottom */

border: thin #990000 solid; /* The border around the boxes at the top and bottom */

/*-----------Link Prefs-----------*/
A:link {
color: #003399; /* This is the color of your basic link*/
text-decoration: none;
font-family: arial, helvetica, geneva
}
A:visited {
color: #003399; /* This is the color of your basic visited link*/
text-decoration: none
}
A:active {
color: #003399; /* This is the color of your active link*/
text-decoration: underline overline; /* Notice you can add or subtract other features */
}
A:hover {
color: #003399; /* This is the color of your hover link*/
text-decoration: underline overline;
font-family: arial, helvetica, geneva
}
