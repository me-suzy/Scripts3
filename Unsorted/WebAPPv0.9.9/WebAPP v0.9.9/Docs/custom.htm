The script comes packaged with three pre-installed Themes (you can download more here).

Standard Theme: The default theme, it is very easy to customize and the help page will walk you through the process step-by-step.

Custom Theme: In this theme you can customize EVERYTHING. Every part of it is defined in the CSS (Style Sheet) allowing you to change the colors of anything you like. This is for the advanced user who knows there way around a Style Sheet.

Graphics Theme: This theme has a lot of graphics to it it - allowing you to change the graphics instead of the code or the style sheet.
