Features:
-Easy to use and install
-Advanced visit log that includes IP address, language, browser, operating system, is user new to site, date and time, last visit, days since last visit, referrer page and keywords used to search your site
-All visit logs are saved on the visits.txt file so to view all your visits only open that file

To install Website Toll Booth Script simply copy the visits.js and visitlog.php files to your website.
Upload the visits.txt to you server and chmod 777 and insert the following line of code in the pages where you want to log new visits. 

<script language="JavaScript" src="visits.js"></script>

Then when you want to review stats, go to your site/visits.txt
