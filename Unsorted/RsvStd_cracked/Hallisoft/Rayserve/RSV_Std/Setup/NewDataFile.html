Create a New Data File

Section 1

Creation of a new data file is split into several steps. Begin by completing the form below then press the "Continue" button.

1.  Room rates per room or per person per room.  Room  Person
2.  Type of room rate matrix.  Seasonal  Daily
3.  Immediate update of reservation table.  Yes No
4.   Percentage of adult rate to apply to children.
5.   The number of different room types.
6.   The number of rate bands in the matrix.  
7.   Minimum lead time in days for reservations.  
8.   Maximum number of nights that can be booked.  
9.   Minimum number of nights that can be booked.  
10.   Date from which reservations can be accepted.      
11.   Date up to which reservations can be accepted.
