The demo applications attempt to show the versatility of Rayserve and the features available.

The first demonstration has all the configuration options turned on and displays the maximum amount of information in the booking form. It uses the seasonal matrix and charges are per room per night with no child discount.

View Demo 1 - The Paradise Hotel

 
The second demonstration has most of the optional configuration options turned off and shows only minimal information in the booking form. It uses the daily matrix and charges are per person per night with child discounts based on children sharing with a set number of adults.

View Demo 2 - The City Hotel
