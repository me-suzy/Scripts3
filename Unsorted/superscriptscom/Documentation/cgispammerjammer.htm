CGI SPAMMER JAMMER    V1.0

Sick of jerks spamming cgi scripts on your site?  Now you can forever block their lame ass with this simple script that scans a parallel logfile for such activity.  Too many support posts from a client?  Solves that problem too!  Just set this up and anyone executing too many scripts is banned from the site.

System Requirements

  • Apache web server
  • Server logfiles
  • .htaccess
  • Perl 5
  • Crontab recommended
  • Ability to run multiple logs recommended

Features

  • Blocks repeated attacks to cgi scripts from the same user instantly
  • Logs attacking addresses so you can counterattack
  • Variable threshold value so you don't accidentally block out innocent users
