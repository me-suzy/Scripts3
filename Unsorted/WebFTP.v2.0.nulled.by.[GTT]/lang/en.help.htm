WebFTP Help

Login Box

Before you can browse the FTP, you must first log in.
To do this, you should just fill the fields below :

User name :

It's the name of your FTP account given by your administrator or hosting service
Beware, remember that some FTP servers are case sensitive, so
username « someuser » and « SomEUsEr » are different.

Password :

It's the password that you chosed or that was given with your username
when you signed up. Remember that some servers are case sensitive, so
password « somepass » and « SOmEpaSs » are different.

Server and port :

This field is not always present, and allows you to chose the FTP
server and port you want to connect to. This can be either a hostname
like « ftp.somehost.com » or an IP address like 123.45.6.789

Folder :

If the field is present, you can enter the home folder you want to
enter to once logged. Else, let the default value.

Timeout :

If this field is present, you can set the timeout value in seconds
before the connection fails and the script interrupts.

Transfert mode :

If present, you can chose the transfert mode to use for download.
ASCII is used only for text files (html, javascript, php) and binary
for pictures, videos and programs. If you don't know, set to binary.

Passive FTP mode :

If present, you can check this box to enable passive FTP mode.
If you can't download files, try to enable passive mode, because some
proxy servers block the incoming connections.

Connect with SSL :

If present, you can check this box to connect to a secure FTP server.
This is only possible if the server has library OpenSSL installed.

Interface buttons

File handling is done with the icons in the action pannel
under the list of files. Here are the descriptions of the buttons :

Delete files :

Once you have selected the files with the checkboxes,
click on this button to delete them. The script will ask you
for a confirmation because non empty folders are also deleted.

Rename files :

Once you have selected the files with the checkboxes,
click on this button to rename them. The script will prompt you the
new names of each file you selected.

Move files :

Once you have selected the files with the checkboxes,
click on this button to move them to another directory. The script will
ask you the name of the target folder, it can be either absolute or
relative. If it doesn't exist, the script attempts to create it.

New Folder :

The script asks you the name of the new folder. It can be
either absolute or relative.

View / Edit file :

This button has an odd behaviour. The first time you click on it,
the checkboxes of the file will turn into radio buttons.
Then you can select the file to view/edit, and click this button again.

Permissions :

Once you have selected the files with the checkboxes,
click on this button to change their permissions.
Script script asks you the octal value for permissions,
see the PHP manual about chmod().

TAR and download :

This button can be disabled by the administrator.
It attempt to download the selected files, and create a TAR
archive with them, and send back the archive to you. The archive
can be compressed with GZIP or BZIP2 if the extensions are presents.

Send a SITE command :

The script asks you the SITE command to send, and try
to execute it on the server. A message will say you if the
command failed or not, but you can't get the server answer.
