Settings
PHP Directory The server path to your directory that will contain Profiler.php
Example: /home/sites/site1/web
PHP URL The url to your directory that will contain Profiler.php
Example: http://www.youdomain.com
Clan Tag Your clan tag which will be automatically added to the beginning of all clan members.
Clan Ranks The different ranks that your clan will use. You can have as many or as little as you like.
Clan Factions The different factions that your clan will use. You can have as many or as little as you like.
Custom Stats The different statistics that your clan will use to profile members. You can have as many or as little as you like.
The checkbox beside causes the form field to be required when signing up a new member.
Background Color 1 & 2 A background color used throughout the script.
Example: #336699
Font Face The font face used throughout the script.
Example: Tahoma
Font Size 1 & 2 The font size used throughout the script.
Example: 2
Font Color The font color used throughout the script.
Example: 000000
Code of Conduct Your clan rules / regulations / requirements.
Use Image Yes or No. Determines whether or not your clan will use a member portrait / image.
Image Directory Name The directory that will contain member portraits / images. This directory is in your PHP directory
Example: images
Image Height The height of your clan portraits / images
Example: 50
Image Width The width of your clan portraits / images
Example: 50
News Date Format The news date format. This field uses the PHP defined date() function. For more information consult the PHP documents.
Example: l F d, Y
News Display Amount The number of news posts displayed as new posts. All other posts are archived
Example: 8
News Format The format you wish to use for each news post. Custom HTML tags are:
<id> - The post ID number
<subject> - The subject of the news post
<date> - The date of the news post
<email> - The email address of the author
<author> - The author of the news post
<news> - The posted news or article
<comments> - The comment count on the news post
News Headlines Format The format you wish to use for each news headline. Custom HTML tags are:
<id> - The post ID number
<subject> - The subject of the news post
<date> - The date of the news post
<email> - The email address of the author
<author> - The author of the news post
<news> - The posted news or article
News Comment Format The format you wish to use for each comment. Custom HTML tags are:
<name> - The name of the comment author
<email> - The email of the comment author
<date> - The date the comment was posted
<comment> - The comment
Ranks Navigation Pannel Yes or No. Displays the navigation pannel on the 'view ranks' page
Ranks Format The format you wish to use for the ranks page. Custom HTML tags are:
<id> - The ID of the member
<alias> - The alias of the member
<email> - The email address of the member
<rank> - The rank of the member
<faction> - The faction of the member
As well as any of your custom statistics. If 'First Name' is a statistic you use then the HTML tag would be <First Name>

Profiles Format The format you wish to use for the profiles page. Custom HTML tags are:
<id> - The ID of the member
<alias> - The alias of the member
<email> - The email address of the member
<rank> - The rank of the member
<faction> - The faction of the member
As well as any of your custom statistics. If 'First Name' is a statistic you use then the HTML tag would be <First Name>

Main Page Format The format you wish to use for the main page. Custom HTML tags are:
<viewnews> - Displays current news posts
