Template File Commands
To insert a command into template.php you need to use standard PHP commands. An example command would be <?php echo $site[text]; ?>

Below is a list of the available commands.
<?php echo $site[text]; ?> - The main body

<?php echo $site[title]; ?> - The title

<?php echo $site[status]; ?> - Displays login status of the current user

<?php echo $site[headlines]; ?> - Current headlines

<?php echo $site[events]; ?> - Current Events

<?php echo $site[top_menu]; ?> - Profiler Navigation (Top Format)

<?php echo $site[side_menu]; ?> - Profiler Navigation (Side Format)

The two navigation styles are designed to give flexibility to website designs. It is recommended that you choose one or the other.
