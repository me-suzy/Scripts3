Administrate User
This feature allows you to change the permission level on clan memebers.
Note: Administrators cannot be edited.
Member The standard user level. These users can only edit their own profiles.
User This user level is intended for trusted clan members. They will have the ability to perform all administrator actions except 'Administrate Users' and 'Edit Settings'
Administrator This user is the highest authority. The user will have access to all script actions and cannot be removed.
