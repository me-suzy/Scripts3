WGS-Expire v1.04 Documentation

 

Server Requirements
 

PHP 4.2.0 or higher

MySQL 3.20.0 or higher

Cron-tab not required

 

Installation and Setup

 

Setting up WGS-Expire software is a simple as 1-2-3! With just 7 easy steps, you can get your script running within less than 10mins.

1.Create a directory called domains (eg, yoursite.com/domains) and chmod 777

2.Unzip all files and upload the file contents to the /domains directory you just created on your server. And Chmod the following:

Chmod 777: member/template, register/template, restore/template and config/template folders

Chmod 666: /domains/domains.php and all template files in /member/template, register/template, restore/template folder and config/template

3. Go to http://www.yoursite.com/domains/config  and edit mysql DB parameters, click ok and install by running http://www.yoursite.com/domains/config/install.php

4.Next, change your admin username and password (Note: Default admin username:admin password:admin)

5.Paypal setup: Go to Paypal Options in admin and edit the variables [Note: IPN URL: http://yoursite.com/domains/ipn.php and return URL: http://yoursite.com/domains/member. Next, login to your paypal account at Paypal.com and go to the Profile tab and click on the Instant Payment Notification link under Selling Preference. Set the IPN url as http://yoursite.com/domains/ipn.php 

6.Cron setup: Male sure cron job path is set as: member/monitor.php and set time interval you want. To start Cron, click "stop" and then click "start" to activate fakecron control.

7.Create membership plan and edit all other variables in admin page
