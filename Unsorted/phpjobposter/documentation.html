Documentation of PHP Job Poster

PHP job poster is completely free of charge.  I'm sorry that the documentation is not that detailed, but I don't really think it is necessary.  This progrma is intended for people who do not have much php or mysql knowledge.  So I didn't really comment my files for editing.

Here are the requirements:

A internet connection
Some place to host these files
A internet mysql database  (you must get this from your isp provider)

13 files included with this package are:

add.php
update.php
delete.php
config.php
admin.php
editdatabase.php
setpass.php
job.php
install.php
createtable.sql
password.html
jobstuff.jpg
documentation.html (this file)

So lets begin setting up you new job poster program.

Step 1.  Unzip the jobposter.zip file.
             If you are reading this you've probably already done that :)

Step 2.  Edit the config.php file in notepad or another text editor.  Edit as follows

              $adminuser="";  ##This is the username of the person you wish to be able to edit your admin file
              $adminpass="";  ##This is the password for the name above


####You do not have to edit this section if you do not want to################
              $maincolor="#F2F1CE";  ##This sets up you color scheme
              $headercolor="#E1AA73";  ##This sets up your color for your headers
              $titlefontcolor="#0000FF";  ##This sets up your color for your fonts
#########end section########################


              $jobtitle="";  #This is the title of the jobs page
              $dbasehost=""; #This is the name of your database host
              $dbaseuser="";  #The username you use to get into your database
              $dbasepassword="";  #The password for the username above
              $dbase="";  #The name of the database
              $folder="";  # The location of these jobposter files  (ie.  http://www.yourserver.com/job/)

Step 2.  Upload all 11 files to a directory on the web.

Step 3.  Run the install.php file by typing in the browser address window  www.yourserver.com/job/install.php  (This file sets up your database.)

Now your database should be set up.

In order to ad jobs to your database you must run the admin.php file.  If you put a name in the $adminuser and $adminpassword variables it will ask you for the username and password.  Once in the admin file you should be able to load, add, update, and delete jobs from the database.  The image below should help.
