Thank you very much for choosing @News Wizard.
Please read this documentation carefully. It contains all of the information which is necessary to install @News Wizard on your server.

The installation of @News Wizard is done completely by the "install.cgi" script.
 
Please note that due to a countless number of different servers and server-setups, the automatic installation process cannot be guaranteed to be successful in all cases. Should your server not execute the installation correctly you will be notified by the installation script that the automatic installation has failed.
In this case please ignore the software bundle "auto-install.tar" and use the "man-install.tar" instead. In our email to you you have received the download links to both packages. Please proceed with the manual installation which is explained in the "documentation-english.htm" of the "man-install.tar" package.

Installation
  • Open only the install.cgi script with your favorite text editor and edit the very first line to reflect the location of PERL on your server.
    By default this line is set to #!/usr/local/bin/perl.
    If the interpreter is at /usr/bin/perl, then this line should be #!/usr/bin/perl. You can get this information with the command "which perl" or "where perl" on the command line once you have telnetted to your account. Check with your system admin if you're not sure.
     
  • Now log on to your web account on your server and create a subdirectory (name of your free choice) in your CGI directory. This is the name which you will later type into the setup form as "Name of CGI Directory" after calling the " install.cgi" script.
    Example: 'nwswiz'
    Set the permission of this directory to full read/write CHMOD 777.
     
  • Copy all cgi-files from your package into this newly created directory.
    Set the permission of all of them to CHMOD 777.
     
  • Now create a subdirectory in your web directory where @News Wizard's HTML files will reside. Give it a name of your choice. This is the name which you will later type into the setup form as
    " @News Wizard Directory" after calling the " install.cgi" script.
    Example: 'nwswiz'
    Set the permission of this directory to full read/write CHMOD 777.
     
  • Copy all image files (GIFs) into this newly created directory.
     
  • Finally call the install.cgi script from your web browser and complete the setup.

Note:
Your server may possibly show different directory names than the ones we describe.
Commonly found names for the CGI directory on many servers are:
'cgi-bin', 'cgi-local' or simply 'bin'.
Your web directory may have one of the following names:
 'public_html', 'www', 'htdocs' or 'httpdocs'
 
That's all you have to do !

The installation form which is displayed to you when you run the install script asks for a minimum of necessary information in order to be able to install your @News Wizard.
If you are not sure about the requested data, please contact your system administrator or host service provider.
You will find some fields already showing a value. You can either accept these (if you used the example-names for your setup) or change them to reflect the situation you have created on your server.

This setup is intended for Unix/Linux type servers only. If you are on a different server like Windows NT you are required to do a manual setup !
