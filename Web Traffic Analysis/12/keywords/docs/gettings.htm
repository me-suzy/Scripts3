setup.php


Before anything else can go ahead, you must customize the file called setup.php. To do this, open it in a plain text editor, such as notepad.
 
This should all be quite straight-forward. The MySQL information should be given to you when your information is sent to you about your account from your websites host, if they provide you with MySQL access.
the database name is in case your host only provides you with one database, or you have to set up each database you want manually. If either of the afformentioned cases are true, change this to your databases name, for example:
  1. I sign up to my host. The information on my account is sent to me, and it has the following included:
    • The MySQL server address is "127.0.0.1"
    • You have been given one database. The database name is "mydb1234".
    • Your database login information is as follows;
    • Username: "user12345"
    • Password: "qwerty"
  2. I change the variables values in setup.php as follows:
    • $my_host = "127.0.0.1";
    • $my_user = "user12345";
    • $my_pass = "qwerty";
    • $my_db = "mydb1234";
  3. I save the setup.php file.

 
This brings us to the following variables in the setup file, which should be pretty obvious:
$username = "Admin";
$password = "qwerty";
This is the login information for the administration page. Make sure you change it before putting these files on your server. Change the words "Admin" and "qwerty" for any other alpha-numerical strings you like, but be sure to keep them contained in quotation marks (" and ")!
