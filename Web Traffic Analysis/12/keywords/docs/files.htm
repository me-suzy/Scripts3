Welcome page! - Getting started - Administration

ref_inst.php

This sets up the MySQL to handle your database. Delete it from your website straight after running it, provided no errors occure.
 

refer.php

This is the keyword logging object. Include it on your main page, and any others you wish. When a search engine hits these pages, it'll do its stuff.
<?php require('refer.php'); ?>
 
Once your all set up, upload all these files to your webserver:
  • admin.php
  • ref_inst.php
  • refer.php
  • setup.php
goto the page "ref_inst.php" to run it and install your keyword logger, then delete it from your webserver if no error messages shown. Now you can find out what keywords are getting you traffic!
