Information About easyPHPstat
easyPHPstat is a web statistic program that does not require the use of a database.

! This is a beta version and may not function 100%. Contact me if you find any errors.

The program registers number of visitors, unique visitors (ip pr. day), color depth, screen size, browser, javascript support, operating systems, country (top domain), keywords from search engines, ip, and referrers.
The program is easy to install. Setup is done by web and includes layout, css and password protection.
! Remember that as this program does not use a database, a large website with many visitors will increase the server's workload.
Recommend a maximum of 1000 unique visitors a day depending on the server speed.
easyPHPstat works with php 4.3 and above. Older versions have not been tested. See F.A.Q for more info.

Installation of easyPHPstat
Unzip easyPHPstat and save it in a directory on the server (e.g. "/easyphpstat").
! Remember to change the file rights in the directory "data/*.*" (chmod 666) - Check your ftp clients helpfile on how to accomplish this.
! Change config.php if needed.
! In the website file you wish to monitor statistics from (e.g. index.html) put the following code in your file. 

<!-- easyPHPstat code -->
<script type="text/javascript">
/*<![CDATA[*/
var ref=""+escape(top.document.referrer); var colord = window.screen.colorDepth;
var res = window.screen.width + "x" + window.screen.height;
document.write("<img src='http://yoursite.com/easyphpstat/stat.php?referer=" + ref + "&amp;js=yes&amp;color=" + colord + "&amp;res=" + res + "' alt='' />");
// ]]>
</script><noscript><div><img src="http://yoursite.com/easyphpstat/stat.php" alt="" /></div>
</noscript>
<!-- easyPHPstat code -->
! Remember to change the 2 <img> tags to match your website (e.g. http://yoursite.com/easyphpstat/stat.php):
Since an image tag "<img>" calls the program, there will be a 1x1pixel transparent gif inserted on the page.
Remember to reset your stats on use to get the correct start date.
To see your stats, use the following url: "http://yoursite.com/easyphpstat".

Setup of easyPHPstat
Setup is done from the web - Go to "Settings" and press "easyPHPstat Setup".
! Username and Password are initially set to "user" / "abc123"

Frequently Asked Questions
Q:
I Get this error: "...Warning: fopen("data/filename", "w")..."?
A:
Change the file permission to "chmode 666" in "/data/*.*"

Q:
Chmod doesn't work?
A:
Some windows ftp clients don't support chmode. Use a program like SmartFtp (can be found at www.download.com)

Q:
The program doesn't register statistics?
A:
Remember to put the JavaScript code on your page and change the path to match your website

Q:
The program shows all hits as "0"?
A:
A few have reported this error when running easyPHPstat on a WinNT/IIS server. The problem is caused by the function "flock()". A way to solve this is removing the function. This may cause problems if 2 people are visiting your website at the exact same time

Q:
My webserver is using a php version below 4.2 - What do I do?
A:
Upgrade... :-)

Q:
I still haven't solved my problem?
A:
Check for updates on the webpage below. If you still have problems contact me. If possible please include your webserver's operating system and php version

Q:
I don't know my webserver's operating system or php version?
A:
Use the php function "phpinfo()". It will tell you all you need to know

Q:
Can I edit this code?
A:
Yes. You can do practically anything you want. Remember to read the license agreement
